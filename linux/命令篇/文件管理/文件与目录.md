## ls

ls命令来自英文单词 `list` 的缩写，中文译为“列出”，其功能是显示目录中的文件及其属性信息，是最常使用的Linux命令之‍一。

### 语法格式

```bash
ls [参数] [文件或目]
```

- 无参数

```bash
zhaohui@ubuntu-26-04-lts:/$ ls
bin   cdrom  etc   lib    lost+found  mnt  proc  run   snap  swap.img  tmp  var
boot  dev    home  lib64  media       opt  root  sbin  srv   sys       usr
```

- -a 显示隐藏文件

```bash
zhaohui@ubuntu-26-04-lts:/$ ls -a
.   bin   cdrom  etc   lib    lost+found  mnt  proc  run   snap  swap.img  tmp  var
..  boot  dev    home  lib64  media       opt  root  sbin  srv   sys       usr
```

- -l 以长格式显示文件的详细信息，包括权限、所有者、大小和修改时间
