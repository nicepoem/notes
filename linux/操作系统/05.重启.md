# shutdown

> `shutdown` 是一个用于**安全地关闭或重启系统**的命令。它会向所有登录用户发送警告消息，并阻止新用户登录，然后按照指定的时间执行关机、重启或进入单用户模式等操作。

## 语法

```bash
shutdown [选项] [时间] [警告消息]
```

### 基本格式

```bash
shutdown [OPTIONS] [TIME] [WALL_MESSAGE]
```

### 选项

| 参数     | 说明                         |
| :------- | :--------------------------- |
| `-c`     | 取消已计划的关机任务         |
| `-h`     | 停止系统（halt），关闭电源   |
| `-r`     | 重启系统（reboot）           |
| `-k`     | 不真正关机，只发送警告消息   |
| `-P`     | 关闭电源（等同于 -h）        |
| `-H`     | 停止系统但不关闭电源（halt） |
| `-f`     | 强制关机/重启，不调用 fsck   |
| `-F`     | 强制在重启时进行 fsck 检查   |
| `-t sec` | 延迟指定秒数后执行           |

### 参数

```
[时间]：        设置多久时间后执行shutdown指令；
[警告信息]：   要传送给所有登入用户的信息。
```

## 实例

指定现在立即关机：

```bash
shutdown -h now
```

指定 5 分钟后关机，同时送出警告信息给登入用户：

```bash
shutdown +5 "System will shutdown after 5 minutes"
```
