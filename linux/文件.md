## cd 切换工作目录

当 Linux 终端(命令行)打开的时候，会默认以用户的 `home` 目录作为当前的工作目。我们可以通过 `cd` 命令，更改当前所在的工作目录。

cd 命令来自英文：Change Directory

- `cd /home` 切换到指定的⽬录

```Bash
[admin@localhost /]$ ls
bin   dev  hello.java  lib    media  opt   root  sbin  sys  usr
boot  etc  home        lib64  mnt    proc  run   srv   tmp  var
[admin@localhost /]$ cd /home
```

- `cd ..` 返回上⼀级⽬录

```Bash
[admin@localhost home]$ ls
admin  test  zwj
[admin@localhost home]$ cd ..
[admin@localhost /]$ 
```

- `cd ~` 切换到当前⽤户的主⽬录

```Bash
[admin@localhost home]$ cd ~
[admin@localhost ~]$ 
```

## pwd 显示当前⽬录

通过 `ls` 来验证当前的工作目录，其实是不恰当的。我们可以通过 `pwd` 命令，来查看当前所在的工作目录。

pwd 命令来自：Print Work Directory

- `pwd` 显示当前⼯作⽬录的绝对路径

```Bash
[admin@localhost home]$ pwd
/home
```

## mkdir 创建新⽬录（文件夹）

英文说明：Make Directory

- `mkdir 目录名称` 创建⼀个新⽬录

- `mkdir 目录名称1 目录名称2 目录名称3` 创建多个⽬录

```

- `mkdir -p 一级目录/二级目录` 创建⼀个新⽬录和⼦⽬录

`-p` 递归创建目录，即如果要创建的目录的父目录不存在，也会一并创建

```

## chmod 修改⽂件权限

Linux 中的权限可以应⽤于三种类别的⽤户：

- ⽂件所有者（u）

- 与⽂件所有者同组的⽤户（g）

- 其他⽤户（o）

<img src="C:/Users/zhaohui/AppData/Roaming/Typora/typora-user-images/image-20260114204513090.png" alt="image-20260114204513090" style="zoom: 50%;" />

### 1、符号模式

符号模式使⽤字⺟来表示权限，如下：

- 读（r）

- 写（w）

- 执⾏（x）
- 所有（a）

例如：

- `chmod u+w file` 给⽂件所有者添加写权限

- `chmod g-r file` 移除组⽤户的读权限
- `chmod o+x file` 给其他⽤户添加执⾏权限
- `chmod u=rwx,g=rx,o=r file` 设置⽂件所有者具有读写执⾏权限，组⽤户具有读执⾏权限，其他⽤户具有

读权限

### 2、数字模式

数字模式使⽤三位⼋进制数来表示权限，每位数字代表不同的⽤户类别（所有者、组、其他⽤户），数字是其各⾃

权限值的总和：

- 读（r）= 4
- 写（w）= 2
- 执⾏（x）= 1

![image-20260114212708403](C:/Users/zhaohui/AppData/Roaming/Typora/typora-user-images/image-20260114212708403.png)

权限模式可以是从 0（⽆权限）到 7（读写执⾏权限）的任何值。

- `chmod 755 file` 使得⽂件所有者有读写执⾏（7）权限，组⽤户和其他⽤户有读和执⾏（5）权限。
- `chmod 644 file` 使得⽂件所有者有读写（6）权限，⽽组⽤户和其他⽤户只有读（4）权限。

使⽤ chmod +x test.sh 或者 chmod 755 test.sh 来给⽂件添加可执⾏权限。

## zip ⽂件压缩

### gzip

1、压缩⽂件

- `gzip filename` 将⽂件压缩成 `.gz` 格式，原⽂件会被压缩后的⽂件替换

```
[admin@localhost ~]$ ls
1.txt  3.txt  pengyuyan.png  模板  图片  下载  桌面
2.txt  java   公共            视频  文档  音乐
[admin@localhost ~]$ gzip 1.txt
[admin@localhost ~]$ ls
1.txt.gz  3.txt  pengyuyan.png  模板  图片  下载  桌面
2.txt     java   公共            视频  文档  音乐
[admin@localhost ~]$ 
```

- `gzip -k filename` 压缩⽂件但保留原⽂件

2、解压⽂件

`gzip -d filename.gz` ：解压 `.gz` 格式的⽂件

```
[admin@localhost ~]$ ls
1.txt.gz  3.txt  pengyuyan.png  模板  图片  下载  桌面
2.txt     java   公共            视频  文档  音乐
[admin@localhost ~]$ gzip -d 1.txt.gz
[admin@localhost ~]$ ls
1.txt  3.txt  pengyuyan.png  模板  图片  下载  桌面
2.txt  java   公共            视频  文档  音乐
[admin@localhost ~]$ 
```

### zip

1、压缩⽂件或⽬录

- `zip -r archive_name.zip directory_or_file` ：压缩⽂件或⽬录到 .zip 格式



2、解压

 `unzip archive_name.zip` 解压 .zip 格式的⽂件



### tar

**tar**

tar 是 Tape ARchive（磁带归档）的缩写，它是一个在Unix/Linux系统中用于将多个文件打包成一个归档文件的工具。虽然名称中有"磁带"，但现在的 `tar` 主要用于文件打包，而不是磁带备份。

- `c` 创建⼀个新的 tar 包

- `z` 使⽤ gzip 压缩
- `v` 显示详细信息
- `f` 指定⽂件名
