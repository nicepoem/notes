## ubuntu配置远程登录

Ubuntu 桌面版通常没有预装 SSH 服务端，需要手动安装。

更新软件包列表

```bash
sudo apt update
```

1. 安装 `openssh-server`

```bash
sudo apt install openssh-server
```

安装完成后，SSH 服务通常会自动启动。

2. 设置开机自启并立即启动

```bash
sudo systemctl enable --now ssh 
```

3. 查看服务状态

```bash
sudo systemctl status ssh
```
